# Đề tài: Đặt lịch khám bệnh

Ứng dụng ASP.NET Web Forms giúp người dùng đăng ký khám bệnh trực tuyến, quản lý lịch hẹn và thông tin bệnh nhân.

## 🔧 Công nghệ
- ASP.NET Web Forms (.NET Framework)
- SQL Server
- Bootstrap 4

## 🚀 Cách chạy
- Mở bằng Visual Studio
- Chạy bằng IIS Express
- Thực thi file `SQL/dangkykhambenh.sql` bằng SSMS để tạo cơ sở dữ liệu

## 📌 Tính năng
- Đăng ký khám bệnh theo chuyên khoa
- Quản lý lịch hẹn
